== Description ==

Adds a canonical link to your blog's home page. As of version 2.9, WordPress adds canonical links automatically, but doesn't place them on the homepage. This adds a link in your header block that helps with search engine optimization.

For more information on canonical links, see [Matt Cutt's blog entry](http://www.mattcutts.com/blog/canonical-link-tag/)
:/
== Installation ==

* Install the widget using Add New from the Plugins menu in the WordPress admin dashboard. Search for Homepage Canonical Link.
* Once activated, this plugin simply injects a canonical link onto your home page, with no further configuration required.
